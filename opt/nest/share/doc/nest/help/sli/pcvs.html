Name:
 pcvs - Convert object to string (long version)
Description:
 pcvs converts every object into a string (long version)
Examples:
 23 pcvs -> (23)
 {3 2 mul dup} pcvs -> ({3 2 mul dup})
 (hello) pcvs -> ((hello))
