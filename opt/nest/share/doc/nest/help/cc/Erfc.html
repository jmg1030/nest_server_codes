Name:
 Erfc - complementary error function
Synopsis:
 x Erfc -> result
Description:
 Computes the error function
             erfc(x) = 1 - erf(x) = 2/sqrt(pi) int_x^inf dt exp(-t^2)
Parameters:
  x (double): error function argument
Examples:
 0.5 erfc -> 0.4795
Author:
 H E Plesser

 Requires the GSL.
