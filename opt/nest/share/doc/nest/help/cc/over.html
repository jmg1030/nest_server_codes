Command: over

NEST HelpDesk Command Index NEST Quick Reference

Name:
 over - Copy object at stack level 1
Synopsis:
 any obj over -> any obj any
Diagnostics:
 Raises StackUnderflow error if there are less than two objects on
  the stack.
Examples:
 1 2 3 over -> 2
1 2 3 4 5 over -> 4
SeeAlso:dup index copy
