Name:
 pprint - pretty print: Print object to a stream
Synopsis:
 ostream any <- -> ostream
Description:
 Alternatives: You can use <-- (undocumented),
  which is the same as pprint.
Examples:
  cerr [1 2 3] pprint endl ; -> [1 2 3]
