Command: sqrt

NEST HelpDesk Command Index NEST Quick Reference

Name:
 sqrt - compute the square root of a non-negative number
Synopsis:
 number sqrt -> double
Description:
 sqrt computes the the square root of a number.
If the value is negative, a RangeCheck error is raised.
Examples:
 4 sqrt -> 2.0
SeeAlso:sqr
